# TPI Bienestar Digital - Instituto Politécnico Formosa 🚀

Este repositorio contiene el desarrollo completo del Trabajo Práctico Integrado sobre **Bienestar Digital**. El proyecto analiza el impacto del consumo de redes sociales en los estudiantes mediante estadística descriptiva, correlación y modelos automatizados de regresión lineal.

## 📂 Estructura del Proyecto

El repositorio cuenta con archivos independientes para cada fase del desarrollo, los cuales deben ejecutarse en orden cronológico:
* **Etapa 1:** Relevamiento inicial y planteamiento del problema.
* **Etapa 4:** Procesamiento descriptivo y construcción de la tabla de frecuencias.
* **Etapa 5:** Modelado matemático preliminar de regresión lineal.
* **Etapa 6:** Unificación, automatización total del análisis y generación de interpretaciones inteligentes.

## 📋 Requisitos Previos

Para poder correr este análisis, necesitas tener instalado **Python** y **Jupyter Notebook** (o usar extensiones compatibles como VS Code o Google Colab).

## 🛠️ Instalación de Librerías

Antes de ejecutar los cuadernos por primera vez, debes instalar las librerías necesarias. Abre tu terminal o consola de comandos y ejecuta el siguiente comando **una sola vez**:

```bash
pip install pandas numpy matplotlib scikit-learn seaborn
```

## 💻 Cómo Ejecutar el Proyecto

1. Asegúrate de tener el archivo de datos `tpi_datos.csv` (o `datos.csv`) en la **misma carpeta** donde se encuentran todos los archivos del notebook (`.ipynb`).
2. Abre tu entorno de **Jupyter Notebook**.
3. Abre el archivo de la etapa que deseas revisar (Etapa 1, 4, 5 o 6).
4. En el menú superior de la interfaz de Jupyter, haz clic en el botón **Run All** (o ve a **Kernel** -> **Restart & Run All**).
5. Repite este mismo paso para cada uno de los archivos si deseas ver la secuencia completa del desarrollo.
